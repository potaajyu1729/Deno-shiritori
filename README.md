# Deno-shiritori

インターンシップ選考課題

## 概要

ブラウザ上で遊べるハイスコア型のしりとりゲームです。
基本的なしりとりのルールに加え、ライフ・制限時間・レベルアップなどのゲーム要素を実装し、繰り返し遊べるよう工夫しました。

## 公開先

https://deno-shiritori-k1tya8t3gsj9.potaajyu.deno.net/

## 実装した機能

### 基本機能（課題要件）

* 直前の単語の表示
* 任意の単語の入力
* 直前の単語の末尾と入力した単語の先頭を比較し、一致した場合のみ単語を更新（不一致の場合はエラーメッセージを表示）
* 「ん」で終わる単語が入力された場合はゲーム終了
* 過去に使用した単語が入力された場合はゲーム終了
* ゲーム中・ゲーム終了後のリセット機能

### 追加実装

ゲーム性を高めるため、以下の機能を実装しました。

* 最初の単語をランダムに決定
* ひらがなのみ入力可能
* ライフ制
* 制限時間
* レベル制（レベルに応じて必要な文字数が増加）
* 実績（アチーブメント）機能
* ハイスコアの保存
* レスポンシブ対応

## AIの活用

デザイン（CSS）の作成には **v0** を活用し、見やすく親しみやすいUIとなるよう調整しました。実装や機能については、自身で設計・開発を行いました。
